el objetivo de este programa es validad y calcular los porcentraje de retorno tomando como ejemplo a una compania de seguros, donde el cliente valida si aplica (ciendo myor de edad)
y que recargos tendra segun sea su estatus, economico, civil etc

asi mismo asumiento que su esposa sera mayor de edad, de lo contrario el programa lo tomara con valor 0 

sin embargo este programa como tal tiene puntos a considerar en mejoras, por ejemplo una grafica amigable con el usuario que le permita entender de mejor manera la funcionalidad el porgrama. 
se podria depurar el codigo y agragar funciones para reducir o optimizar las lineas de codigos.
